# AI Study Assistant

AI Study Assistant is a simple project designed to help students learn faster using AI tools.

## Features

- Summarize long study notes
- Explain difficult concepts in simple language
- Generate study plans for exams
- Help students revise quickly

## Goal

The goal of this project is to make learning easier for students by using AI-powered tools.

## Future Improvements

- Add AI chatbot
- Add voice explanation
- Create web interface
